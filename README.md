# Food Wastage Prevention

This repository contains both the frontend and backend for the Food Wastage Prevention project.

## Structure
- frontend/ - React app
- backend/ - Node.js + Express API

## Quick Start
1. Install dependencies:
   - npm run install:frontend
   - npm run install:backend
2. Start the apps:
   - npm run frontend
   - npm run backend

## Deployment
- Frontend: deploy the `frontend/` folder to Vercel
- Backend: deploy the `backend/` folder to Vercel
